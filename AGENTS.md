# Retail Agent Specifications: "PromoGuard"
Role: Automated Discount & Integrity Agent
Objective: Monitor the shopping cart and apply "Loyalty Tier" discounts while preventing coupon stacking.

Rules:
1. If the cart total > $500, apply a 15% 'VIP Retail' discount.
2. If an item is from the 'Electronics' category, ensure the 'Eco-Tax' of $5 is added.
3. Prevent users from applying more than one manual discount code.
